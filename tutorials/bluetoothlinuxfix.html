How to fix bluetooth not working on linux

Press CTRL + ALT + T, to open a terminal window

Commands

On Ubuntu:

sudo apt install bluez* -1

On Fedora:

sudo dnf install bluez bluez-tools

On Manjaro:

sudo pacman -S bluez bluez-utils

And if there is anything else here are some further steps

Install Blueman (Optional):
Blueman is an independent project that provides a user-friendly front-end for BlueZ. You can install it separately using the following command:
sudo apt install blueman # For Ubuntu
sudo dnf install blueman # For Fedora
sudo pacman -S blueman # For Manjaro

Start the Bluetooth Daemon:
To have Bluetooth available each time you boot your computer, enable and start the bluetooth.service daemon. Use the following commands:
sudo systemctl enable bluetooth.service
sudo systemctl start bluetooth.service
