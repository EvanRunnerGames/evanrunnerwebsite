How to disable bitlocker on windows

STEP 1

Run powershell as administraior and paste this command

manage-bde -status

The output should look something like this the red is my external hard drive/usb
the green is my internal hard drive and the white is everything else

BitLocker Drive Encryption: Configuration Tool version 10.0.22621
Copyright (C) 2013 Microsoft Corporation. All rights reserved.

Disk volumes that can be protected with
BitLocker Drive Encryption:
Volume C: []
 [OS Volume]

    Size: 186.26 GB
    BitLocker Version: None
    Conversion Status: Fully Encrypted
     Percentage Encrypted: 0.0%
    Encryption Method: None
    Protection Status: Protection Off
    Lock Status: Unlocked
    Identification Field: None
    Key Protectors: None Found
Volume D: [Linux Mint 21_3 Cinnamon 64-bit]
[Data Volume]

    Size: 1816.44 GB
    BitLocker Version: None
    Conversion Status: Fully Decrypted
    Percentage Encrypted: 0.0%
    Encryption Method: None
    Protection Status: Protection Off
    Lock Status: Unlocked
    Identification Field: None
    Automatic Unlock: Disabled
    Key Protectors: None Found

 PS C:\Windows\system32>

So the one that is not your USB is your internal hard drive/disk so right there where it says volume and a letter I have it as orange look at yours and that is your internal hard drive's volume letter.

And where it says fully encrypted is yellow, we need to change that to fully decrypted so get that volume's letter.

And run this command:
Disable-BitLocker -MountPoint "C:"

For me, it is C:, but for you, it might be something different so check your internal hard drive's volume letter.

manage-bde -off "C:"

And after that, run manage-bde -off "C:"
To check if it is fully decrypted. If it is not, wait and check again later.

After that, BitLocker protection is off.
