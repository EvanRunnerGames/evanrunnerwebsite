This is a document for all the linux commands and there uses

Press CTRL + ALT + T, to open a terminal window

Commands

1. `ls` - List directory contents.
2. `cd` - Change the current directory.
3. `pwd` - Print working directory.
4. `mkdir` - Make directories.
5. `rmdir` - Remove empty directories.
6. `rm` - Remove files or directories.
7. `cp` - Copy files or directories.
8. `mv` - Move or rename files or directories.
9. `touch` - Change file timestamps or create empty files.
10. `chmod` - Change file mode bits.
11. `chown` - Change file owner and group.
12. `ln` - Make links between files.
13. `find` - Search for files in a directory hierarchy.
14. `grep` - Print lines matching a pattern.
15. `sed` - Stream editor for filtering and transforming text.
16. `awk` - Pattern scanning and processing language.
17. `cat` - Concatenate and print files.
18. `tail` - Output the last part of files.
19. `head` - Output the first part of files.
20. `sort` - Sort lines of text files.
21. `uniq` - Report or omit repeated lines.
22. `diff` - Compare files line by line.
23. `tar` - Tape archive; an archiving utility.
24. `gzip`, `gunzip` - Compression and decompression using Lempel-Ziv coding (LZ77).
25. `bzip2`, `bunzip2` - A block-sorting file compressor, v1.0.
26. `xz`, `unxz` - Compress or decompress .xz and .lzma files.
27. `df` - Report file system disk space usage.
28. `du` - Estimate file space usage.
29. `free` - Display amount of free and used memory in the system.
30. `top` - Display Linux processes.
31. `ps` - Report a snapshot of the current processes.
32. `kill` - Send a signal to a process.
33. `sudo` - Execute a command as another user.
34. `man` - An interface to the system reference manuals.
35. `echo` - Display a line of text.
36. `which` - Locate a command.
37. `ssh` - OpenSSH SSH client (remote login program).
38. `scp` - Secure copy (remote file copy program).
39. `wget` - The non-interactive network downloader.
40. `curl` - Transfer data from or to a server.
41. `nano`, `vim`, `emacs` - Text editors.
42. `screen`, `tmux` - Terminal multiplexers.
43. `ip` - Show / manipulate routing, devices, policy routing, and tunnels.
44. `netstat` - Print network connections, routing tables, interface statistics, masquerade connections, and multicast memberships.
45. `ping` - Send ICMP ECHO_REQUEST to network hosts.
46. `systemctl` - Control the systemd system and service manager.
47. `journalctl` - Query and display messages from the journal.
